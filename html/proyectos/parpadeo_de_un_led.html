Parpadeo de un LED

Descripción

El parpadeo de un LED con Arduino Uno implica escribir un programa simple que alternará entre encender y apagar un pin digital conectado al LED. Esto se logra utilizando la función digitalWrite() para establecer el estado del pin como HIGH (encendido) o LOW (apagado), y la función delay() para controlar la duración de cada estado.

Materiales necesarios:

  • Arduino Uno (u otra placa compatible)
  • LED
  • Resistencia de 220 ohmios
  • Protoboard y cables de conexión

Instrucciones:

  1. Conecta el LED a Arduino Uno: el lado largo del LED (ánodo) va al pin digital 13 y el lado corto (cátodo) a tierra (GND) a través de una resistencia de 220 ohmios.
  2. Conecta Arduino Uno a tu computadora usando un cable USB.
  3. Abre el software Arduino IDE en tu computadora
  4. Escribe el siguiente código en Arduino IDE y cargale
  5. Disfruta de tu primer programa

Esquema de conexión

Código de Ejemplo (para Arduino IDE):

Copied

/*

  https://alvelectronics.com
*/
const int ledPin = 13; //Asigna el numero de pin a la variable ledPin

void setup() {
  pinMode(ledPin, OUTPUT);  //Establece al pin 13  como salida
}

void loop() {
  digitalWrite(ledPin, HIGH);  //Pone en estado Alto al led
  delay(1000);                 //Pausa de 1 segundo
  digitalWrite(ledPin, LOW);   //Pone en estado Bajo al led
  delay(1000);                 //Pausa de 1 segundo
}
